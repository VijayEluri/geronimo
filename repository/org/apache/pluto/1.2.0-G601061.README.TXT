How to build Pluto 1.2.0 (w/ local modification to disable throwing of PortletSecurityException) for Geronimo:

Checkout pluto 1.2.0
  svn co -r 601061 https://svn.apache.org/repos/asf/portals/pluto/trunk pluto

  patch -Np0 -i geronimo-3855.patch
  mvn install
