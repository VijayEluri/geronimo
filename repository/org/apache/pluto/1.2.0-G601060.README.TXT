Private Build of Pluto for Geronimo.										       

How to build Pluto 1.2.0 (no local modification) for Geronimo:

Checkout pluto 1.2.0
  svn co   https://svn.apache.org/repos/asf/portals/pluto/trunk  pluto

Build pluto
  cd pluto
  mvn install
